### Week 5 Objectives:

* Understand and practice advanced file server management features such as data deduplication, shadow copies, and user quotas.
* Learn how to scale storage and throughput capacity to optimize performance in a cloud environment.
* Practice Amazon S3 bucket management including static website hosting, public access configuration, and versioning.
* Explore Amazon CloudFront setup for content delivery and understand caching behavior.
* Gain knowledge of cross-region replication and AWS Shared Responsibility Model.
* Understand the basics of AWS Identity and Access Management (IAM) and its role in securing cloud resources.
