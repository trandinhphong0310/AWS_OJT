### Week 6 Objectives:

* Understand and implement **AWS identity and access management** concepts using IAM, Roles, and Policies.  
* Learn how to manage **user authentication and authorization** through **Amazon Cognito** and **AWS Organizations**.  
* Explore **AWS Security Hub** for centralized security monitoring and compliance checks.  
* Practice using **AWS Key Management Service (KMS)** for data encryption and protection.  
* Automate EC2 instance management using **AWS Lambda** and **AWS CLI**.  
* Apply **resource tagging** best practices to organize and manage AWS resources efficiently.
